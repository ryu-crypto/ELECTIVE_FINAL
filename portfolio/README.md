# Personal Portfolio Website

## Project Overview
This project is a responsive personal portfolio website designed to showcase my biography, technical skills, and a collection of my academic and personal projects. It serves as a digital resume and demonstrates mastery of web structure (HTML), styling (CSS/Bootstrap), and modern deployment workflows.

## Technology Stack
- **HTML5**: Semantic structure of the website.
- **CSS3**: Custom styling and layout adjustments.
- **Bootstrap 5**: Responsive grid system and components.
- **Git/GitHub**: Version control and deployment.

## Features
- **Hero Section**: Introduction with a call to action.
- **About Me**: Professional biography and photo.
- **Skills**: Visual representation of technical skills.
- **Projects Gallery**: Showcase of 3 projects (Project 1, Project 2, Project 3).
- **Contact**: Form and social media links.
- **Responsive Design**: Optimized for mobile, tablet, and desktop devices.

## How to View
1. Clone the repository.
2. Open `index.html` in your web browser.
